# LAP 3 24 hour Assignment

## Installation & Usage

### Instalation
- Clone or download the repo.
- Open your terminal and navigate to `GitHub` folder.
- Run `npm install` to install dependencies.
### Usage
- Run `npm run dev` to launch the app.
